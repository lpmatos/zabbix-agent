# zabbix-agent

Script that install and configure Zabbix Agent on Linux or Windows

### Commands

```bash
tail -f /var/log/zabbix/zabbix_agentd.log
/etc/init.d/zabbix-agent restart
zabbix_get -s 172.31.92.92 -p 10050 -k agent.hostname
```
